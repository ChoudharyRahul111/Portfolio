# Rahul Choudhary - Personal Portfolio

A professional, responsive portfolio website built to showcase my skills, education, and passion for Cloud Computing. 

## Features
- **Responsive Design**: Adapts beautifully across desktops, tablets, and mobile devices.
- **Modern UI/UX**: Implements glassmorphism effects, a dynamic animated background (stars), and clean typography.
- **Structured Sections**:
  - **Home**: A brief introduction and quick navigation links.
  - **About**: Details about my interest in modern web technologies, scalable architecture, and cloud infrastructure.
  - **Education**: A timeline layout highlighting my academic journey (BTEUP, AKTU, etc.).
  - **Contact**: A clean form and details to get in touch for opportunities.

## Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with custom properties, flexbox/grid layouts, and animations.
- **JavaScript**: For interactive elements like the mobile navigation menu.
- **External Resources**: 
  - FontAwesome (Icons)
  - Google Fonts (Inter, Outfit)

## Getting Started

To view the project locally, you don't need any special build tools.

1. Clone this repository:
   ```bash
   git clone <your-github-repo-url>
   ```
2. Navigate into the project directory:
   ```bash
   cd Portfolio
   ```
3. Open the `index.html` file in your preferred web browser. 
   - *Tip: If you are using Visual Studio Code, you can use the "Live Server" extension for a better development experience.*
